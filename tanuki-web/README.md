# Talking Tanuki - Marketing Website

This is the marketing website for Talking Tanuki, built with Next.js.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) to view the website.

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: TailwindCSS (to be added)
- **Deployment**: Vercel (recommended)
